COMP 2156 Devops Course
 DevOps is a set of practices that combine software development (Dev) and IT operations (Ops).
 A course on DevOps would cover the principles, methods, and tools used in this approach.
 DevOps courses often include hands-on labs or projects.
 Students may need a foundational understanding of software development, and networking concepts.
 The course could be offered in various formats, such as lectures, labs, workshops, and possibly online components for remote learning.
 Devops covers understanding version control concepts and using tools like Git for managing source code changes, branching, and merging.
 Covers introduction to CI principles and tools like Jenkins, Travis CI, or GitLab CI for automating the build and testing of applications.
